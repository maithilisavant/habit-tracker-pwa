# PWA Icons

This app requires the following icon files for PWA functionality:

- `pwa-192x192.png` - 192x192 pixels (required)
- `pwa-512x512.png` - 512x512 pixels (required)
- `apple-touch-icon.png` - 180x180 pixels (for iOS)

## Quick Icon Generation

You can create simple icons using any image editor or online tool:

1. Create a square image with your app logo/icon
2. Export at the required sizes
3. Place them in the `public/` directory
